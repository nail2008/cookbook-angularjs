P39 使用过滤器格式化数据

过滤器的语法： { { expression | filterName : parameter1 : ...parameterN } }

例1：货币 { { 12.9 | currency } }

{{ 12.9|currency }}

例2：保留整数的人民币(书上的写法是错的) { { 12.9 | number : 0 | currency : '￥' } }

{{ 12.9 | number : 0 | currency : '￥' }}

例3：全部大写 { { 'behold the majesty of your page title' | uppercase } }

{{ 'behold the majesty of your page title' | uppercase }}
